# 5to. Congreso Internacional de Procesamiento de Lenguaje Natural para las Lenguas Indígenas

Desarrollo de programa para la identificación de tópicos en documentos en lengua índigena purépecha,
utilizando el algoritmo de aprendizaje automático K-means. Galardonado con el 1er lugar en la
categoría de ponencias en el congreso PLN Indígenas.
